🎯 LeetCode 精讲 #3 - 无重复字符的最长子串

📋 题目描述

来源: LeetCode 第3题
难度: Medium
链接: https://leetcode.cn/problems/longest-substring-without-repeating-characters/

题目原文

给定一个字符串 s，请你找出其中 不含有重复字符 的 最长子串 的长度。

示例

示例 1:

输入: s = "abcabcbb"
输出: 3
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

示例 2:

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3:

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是子串的长度，"pwke" 是一个子序列，不是子串。

约束条件

  • 0 <= s.length <= 5 * 10^4
  • s 由英文字母、数字、符号和空格组成

💡 解题思路

核心思想：滑动窗口 (Sliding Window)

这道题是滑动窗口算法的经典入门题。滑动窗口的本质是维护一个动态区间，通过移动左右边界来寻找满足条件的子串/子数组。

滑动窗口的核心要素

  1. 窗口定义：一个连续的子串区间 [left, right]
  2. 窗口性质：窗口内的字符互不相同（无重复）
  3. 扩张：右指针右移，将新字符纳入窗口
  4. 收缩：当发现重复字符时，左指针右移，直到窗口恢复"无重复"性质

详细执行过程

以 s = "abcabcbb" 为例：

初始: 窗口为空, left=0, max_len=0

步骤1: right=0, s[0]='a', 窗口="a", 无重复
       max_len=1
       
步骤2: right=1, s[1]='b', 窗口="ab", 无重复
       max_len=2
       
步骤3: right=2, s[2]='c', 窗口="abc", 无重复
       max_len=3
       
步骤4: right=3, s[3]='a', 'a'已在窗口中(位置0)
       收缩: left移动到位置1 (原'a'的下一个)
       窗口="bca", max_len=3
       
步骤5: right=4, s[4]='b', 'b'已在窗口中(位置1)
       收缩: left移动到位置2
       窗口="cab", max_len=3
       
步骤6: right=5, s[5]='c', 'c'已在窗口中(位置2)
       收缩: left移动到位置3
       窗口="abc", max_len=3
       
步骤7: right=6, s[6]='b', 'b'已在窗口中(位置5)
       收缩: left移动到位置6
       窗口="b", max_len=3
       
步骤8: right=7, s[7]='b', 'b'已在窗口中(位置6)
       收缩: left移动到位置7
       窗口="b" (新), max_len=3

最终结果: 3

关键点：如何快速判断字符是否在窗口中？

使用哈希集合 (HashSet) 或 哈希映射 (HashMap)：

  • Set方案：只记录字符是否存在，O(1)判断
  • Map方案：记录字符→最新位置的映射，可以快速定位left应该移动到哪里

📝 参考代码

方法一：哈希集合 (HashSet)

def lengthOfLongestSubstring(s: str) -> int:
    """
    滑动窗口 + 哈希集合
    时间复杂度: O(n) - 每个字符最多被访问两次
    空间复杂度: O(min(m, n)) - m为字符集大小
    """
    char_set = set()  # 记录窗口内的字符
    left = 0          # 左指针
    max_len = 0       # 最大长度
    
    for right in range(len(s)):
        # 如果当前字符已在窗口中，收缩左边界直到无重复
        while s[right] in char_set:
            char_set.remove(s[left])
            left += 1
        
        # 将当前字符加入窗口
        char_set.add(s[right])
        
        # 更新最大长度
        max_len = max(max_len, right - left + 1)
    
    return max_len

方法二：哈希映射 (HashMap) - 优化版

def lengthOfLongestSubstring_v2(s: str) -> int:
    """
    滑动窗口 + 哈希映射（优化版本）
    核心优化: 当发现重复时，直接跳到重复字符的下一个位置
    时间复杂度: O(n)
    空间复杂度: O(min(m, n))
    """
    char_index = {}   # 字符 -> 最新索引的映射
    left = 0          # 左指针
    max_len = 0       # 最大长度
    
    for right in range(len(s)):
        # 如果当前字符在窗口内，直接跳到重复位置的下一个
        # 注意: 需要用max确保left不会回退
        if s[right] in char_index and char_index[s[right]] >= left:
            left = char_index[s[right]] + 1
        
        # 更新当前字符的最新位置
        char_index[s[right]] = right
        
        # 更新最大长度
        max_len = max(max_len, right - left + 1)
    
    return max_len

def compare_methods():

方法三：数组优化（针对ASCII字符集）

def lengthOfLongestSubstring_v3(s: str) -> int:
    """
    针对ASCII字符的极致优化版本
    使用数组代替哈希表，O(1)访问且常数更小
    """
    # 记录每个字符最后一次出现的位置，-1表示未出现过
    # 假设字符集为ASCII (128个字符) 或扩展ASCII (256个字符)
    last_pos = [-1] * 128
    left = 0
    max_len = 0
    
    for right in range(len(s)):
        char_code = ord(s[right])
        
        # 如果字符在窗口内，收缩左边界
        if last_pos[char_code] >= left:
            left = last_pos[char_code] + 1
        
        # 更新字符位置
        last_pos[char_code] = right
        
        # 更新最大长度
        current_len = right - left + 1
        if current_len > max_len:
            max_len = current_len
    
    return max_len

⏱️ 复杂度分析

方法 时间复杂度 空间复杂度 说明
暴力枚举 O(n³) O(min(m,n)) 枚举所有子串，检查是否重复
哈希集合 O(n) O(min(m,n)) 每个字符最多被访问2次
哈希映射 O(n) O(min(m,n)) 跳过了重复字符间的所有位置
数组优化 O(n) O(1) 固定128/256空间，常数最小

符号说明：

  • n = 字符串长度
  • m = 字符集大小（如ASCII为128，Unicode更大）

为什么时间复杂度是O(n)？

虽然看起来有两层循环（for + while），但left和right指针都只会从左向右移动，每个字符最多被访问两次（入窗口一次，出窗口一次），因此总操作次数是线性的。

🔑 关键要点

1. 滑动窗口的适用场景

滑动窗口适用于寻找满足某种条件的连续子串/子数组：

  • 最大/最小长度
  • 特定和/积
  • 包含特定字符集合
  • 无重复/单调性等约束

2. 窗口收缩的两种策略

策略 适用场景 代码特点
逐步收缩 需要保持某种累计性质 while循环逐个移除
跳跃收缩 可以直接定位新边界 left = index[char] + 1

3. 本题易错点

⚠️ 易错点1: Map方案中忘记判断char_index[s[right]] >= left

# 错误: 直接赋值，可能把left拉回已经过期的位置
if s[right] in char_index:  # 缺少位置判断！
    left = char_index[s[right]] + 1

⚠️ 易错点2: 混淆子串和子序列

  • 子串 (Substring): 必须连续
  • 子序列 (Subsequence): 不必连续 本题要求的是子串，不是子序列。

⚠️ 易错点3: 边界处理

  • 空字符串返回0
  • 单字符返回1
  • 全部相同字符返回1

4. 滑动窗口模板

def sliding_window_template(s):
    from collections import defaultdict
    
    window = defaultdict(int)  # 窗口状态
    left = 0
    result = 0
    
    for right in range(len(s)):
        # 扩大窗口：纳入s[right]
        window[s[right]] += 1
        
        # 收缩窗口：直到满足条件
        while not is_valid(window):
            window[s[left]] -= 1
            left += 1
        
        # 更新答案
        result = max(result, right - left + 1)
    
    return result
