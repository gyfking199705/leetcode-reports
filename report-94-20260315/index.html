🎯 LeetCode 精讲 #94 - 二叉树的中序遍历

📋 题目描述

原题链接: https://leetcode.cn/problems/binary-tree-inorder-traversal/

难度: 简单
标签: 栈、树、深度优先搜索、二叉树

题目原文

给定一个二叉树的根节点 root，返回它的 中序遍历 。

示例

示例 1：

输入：root = [1,null,2,3]
输出：[1,3,2]

示例 2：

输入：root = []
输出：[]

示例 3：

输入：root = [1]
输出：[1]

约束条件

  • 树中节点数目在范围 [0, 100] 内
  • -100 <= Node.val <= 100

进阶: 递归算法很简单，你可以通过迭代算法完成吗？

💡 解题思路

🔍 什么是中序遍历？

二叉树的遍历方式有三种（按根节点访问顺序）：

遍历方式 顺序 记忆口诀
前序遍历 根 → 左 → 右 根左右
中序遍历 左 → 根 → 右 左根右
后序遍历 左 → 右 → 根 左右根

中序遍历的核心特征：对于二叉搜索树(BST)，中序遍历的结果是有序的（升序）！

        1
         \
          2
         /
        3

中序遍历过程：
1. 访问左子树 (1没有左子树，访问根1)
2. 访问根节点 (1) → 输出 [1]
3. 访问右子树 (2)
   - 2的左子树是3
   - 访问3的左子树 (没有)
   - 访问3 (根) → 输出 [1, 3]
   - 访问3的右子树 (没有)
   - 访问2 (根) → 输出 [1, 3, 2]
   - 访问2的右子树 (没有)

最终结果: [1, 3, 2]

📝 解法一：递归（最直观）

核心思想

按照"左 → 根 → 右"的顺序递归访问每个节点。

代码实现 (Python)

# Definition for a binary tree node.
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

class Solution:
    def inorderTraversal(self, root: TreeNode) -> list[int]:
        result = []
        
        def inorder(node):
            if not node:
                return
            # 1. 递归访问左子树
            inorder(node.left)
            # 2. 访问根节点
            result.append(node.val)
            # 3. 递归访问右子树
            inorder(node.right)
        
        inorder(root)
        return result

        └─> result.append(3)      [4, 2, 5, 1, 3]
        └─> inorder(None) 返回

最终结果: [4, 2, 5, 1, 3]

📝 解法二：迭代（显式栈）

为什么需要迭代？

递归虽然简洁，但有以下问题：

  1. 空间复杂度：递归调用栈深度可能达到O(n)
  2. 栈溢出风险：极端情况下（链状树）可能栈溢出
  3. 面试要求：题目进阶明确要求用迭代

核心思想

用显式栈模拟递归调用栈，手动控制访问顺序。

迭代策略：左链入栈法

核心思路：
1. 一直向左走，沿途节点入栈
2. 走到尽头时，出栈访问节点
3. 转向右子树，重复步骤1-2

代码实现 (Python)

class Solution:
    def inorderTraversal(self, root: TreeNode) -> list[int]:
        result = []
        stack = []
        current = root
        
        while current or stack:
            # 步骤1: 一路向左，沿途入栈
            while current:
                stack.append(current)
                current = current.left
            
            # 步骤2: 出栈访问节点
            current = stack.pop()
            result.append(current.val)
            
            # 步骤3: 转向右子树
            current = current.right
        
        return result

迭代执行过程图解

        1
       / \
      2   3
     / \
    4   5

初始: current = 1, stack = [], result = []

第1轮:
  - 向左链入栈: stack = [1, 2, 4], current = None
  - 出栈4, result = [4]
  - 转向右: current = None

第2轮:
  - current为None, 出栈2, result = [4, 2]
  - 转向右: current = 5

第3轮:

  - 转向右: current = 3

第5轮:
  - 向左链入栈: stack = [3], current = None
  - 出栈3, result = [4, 2, 5, 1, 3]

第6轮:
  - stack为空, current为None, 结束

最终结果: [4, 2, 5, 1, 3] ✓

📝 解法三：Morris遍历（常数空间）

核心思想

利用树的空闲指针（叶子节点的左右子树指针）建立临时链接，避免使用栈。

算法步骤

当前节点cur，只要cur不为空：
1. 如果cur.left为空：
   - 访问cur
   - cur = cur.right

2. 如果cur.left不为空：
   - 找到cur左子树的最右节点pre（前驱节点）
   - 如果pre.right为空：
     * pre.right = cur （建立线索）
     * cur = cur.left
   - 如果pre.right等于cur：
     * 访问cur
     * pre.right = None （恢复树结构）
     * cur = cur.right

代码实现 (Python)

class Solution:
    def inorderTraversal(self, root: TreeNode) -> list[int]:
        result = []
        current = root
        
        while current:
            if not current.left:
                # 情况1: 没有左子树，直接访问
                result.append(current.val)
                current = current.right
            else:
                # 情况2: 有左子树，找前驱节点
                pre = current.left
                while pre.right and pre.right != current:
                    pre = pre.right
                
                if not pre.right:
                    # 第一次访问，建立线索
                    pre.right = current
                    current = current.left
                else:
                    # 第二次访问，恢复树结构
                    result.append(current.val)
                    pre.right = None
                    current = current.right
        
        return result

📊 复杂度分析

解法 时间复杂度 空间复杂度 特点
递归 O(n) O(h) 栈空间 最简洁，h为树高度
迭代 O(n) O(h) 显式栈 避免递归栈溢出
Morris O(n) O(1) 破坏/恢复树结构

注: n为节点数，h为树高度。最坏情况下（链状树）h = n。

💡 面试高频考点

  1. 三种遍历方式的区别？

    • 前序：根左右 —— 复制树、序列化
    • 中序：左根右 —— BST有序输出
    • 后序：左右根 —— 删除树、计算高度

  2. 递归 vs 迭代的本质区别？

    • 递归用系统调用栈，迭代用显式栈
    • 两者逻辑完全一致，只是栈的管理方式不同

  3. Morris遍历的优缺点？

    • 优点：O(1)空间，不需要额外内存
    • 缺点：会临时修改树结构，不适用于多线程/只读场景

  4. BST的中序遍历有什么特殊性质？

    • 结果一定是升序排列！这是解决很多BST问题的基础
